What is GitHub, Git, HTMl, CSS, and JavaScript

GitHub - GitHub is a for-profit company that offers a cloud-based Git repository hosting service. Essentially, it makes it a lot easier for individuals and teams to use Git for version control and collaboration.

Git - Git is a distributed version control system , which means that the entire codebase and history is available on every developer's computer, which allows for easy branching and merging.

HTML - Hypertext Markup Language

CSS - Cascading Style Sheets

JavaScript - Javascript is used by programmers across the world to create dynamic and interactive web content like applications and browsers
